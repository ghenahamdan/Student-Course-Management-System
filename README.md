# Student Course Management System
Java application for managing students, courses, enrollments, and grades.

## Features
- Add/view students & courses
- Enroll students with duplicate/capacity checks
- Assign and view grades
- Thread-safe concurrent enrollment simulation
- Stream analytics (sort, filter, count)

## Quick Start
1. Clone the repo
2. Open in IntelliJ
3. Run `Main.java`

## Menu Options
| Option | Description |
|--------|-------------|
| 1-2 | Add student/course |
| 3 | Enroll student |
| 4-5 | View all |
| 6-8 | Stream analytics |
| 9-10 | Grades |
| 11 | Thread simulation |
| 0 | Exit |

## Sample Data
Pre-loaded with 4 students and 3 courses with grades.
